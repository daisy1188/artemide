# USER GUIDE
## Avvio
1. `cd pdf-overlay-manager/app`
2. `py -m http.server 8000`
3. Apri `http://localhost:8000`

## Workflow rapido
- Apri PDF da topbar.
- Disegna Box/Note/Measure/Crop/Exclude da toolbar.
- Seleziona e modifica da pannello Properties.
- Gestisci layer in pannello Layers.
- Cerca testo in Text Search.
- Esporta JSON/CSV/PNG da topbar.

## Shortcut
- `1..7` tool rapidi
- `Del` elimina selezione
- `Esc` torna a Select
- `Ctrl+Z`, `Ctrl+Y`, `Ctrl+Shift+Z` undo/redo
